# 产品需求文档 — 截屏识字幕

## 项目概述

一款运行在 Windows 上的录屏/截屏软件，支持声音录制、离线语音识别生成字幕。UI 采用可爱风格，面向非技术用户。

## 用户画像

- 不懂代码的普通用户
- 需要录制屏幕内容并配上字幕
- 偏爱可爱、温暖的界面风格

## 用户操作流程

### 截屏流程
```
打开软件 → 选择全屏/区域模式 → 点击"截屏"按钮
  → [区域模式] 弹出框选遮罩 → 拖拽框选 → 截图保存 → 列表中显示
  → [全屏模式] 直接截图保存 → 列表中显示
```

### 录屏流程
```
打开软件 → 选择全屏/区域模式 → 开关音源 → 点击"录屏"按钮
  → [区域模式] 弹出框选遮罩 → 框选 → 开始录制
  → [全屏模式] 直接开始录制
  → 显示计时器 + 呼吸灯 → 点击停止 → 视频保存 → 列表中显示
```

### 字幕生成流程
```
在列表中选择视频 → 点击"生成字幕"
  → 显示进度 → 等待完成 → 字幕文件生成 → 可点击"导出字幕"
```

---

## 边界条件

- 录制最长时长：无限制（受磁盘空间约束）
- 存储路径：`%USERPROFILE%\Videos\ScreenRecorder\`（视频）和 `%USERPROFILE%\Pictures\ScreenRecorder\`（截图）
- 支持的导出字幕格式：SRT
- 视频格式：MP4 (H.264)
