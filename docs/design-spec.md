# UI 设计规范文档 — 截屏识字幕

## 设计理念

**关键词**：可爱、温暖、柔软、治愈、圆润

整体风格参考日系可爱风（Kawaii），用柔和的粉色系搭配清新的绿色系，营造温馨舒适的视觉体验。

---

## 配色方案

| 色值 | 名称 | 用途 |
|------|------|------|
| `#fef3ea` | 奶油白 | 页面主背景色 |
| `#cc7d7d` | 豆沙红 | 主按钮、重要操作、强调元素 |
| `#f5b4ae` | 浅粉 | 按钮悬浮态、次要按钮、选中高亮 |
| `#19645c` | 深绿 | 正文文字、图标、标题 |
| `#84cccc` | 青蓝 | 进度条、激活态、链接 |
| `#c4e4dd` | 浅绿 | 卡片背景、列表项背景 |
| `#d8eef1` | 淡蓝 | 微妙背景高亮、分隔区域 |

### 色值速查（CSS 变量）

```css
:root {
  --color-bg:       #fef3ea;
  --color-primary:  #cc7d7d;
  --color-secondary:#f5b4ae;
  --color-text:     #19645c;
  --color-accent:   #84cccc;
  --color-card:     #c4e4dd;
  --color-highlight:#d8eef1;
}
```

## 排版规范

### 字体

| 用途 | 字体 | 备选 |
|------|------|------|
| 标题 | 站酷快乐体 / 猫啃什锦黑 | "Microsoft YaHei", sans-serif |
| 正文 | 思源柔黑体 / ZCOOL KuaiLe | "Microsoft YaHei", sans-serif |
| 等宽 | Consolas, monospace | — |

> 可爱字体文件放入 `assets/fonts/`，通过 `@font-face` 引入。
> 如字体文件过大，优先使用系统自带中文字体 + CSS 模拟圆角效果。

### 字号

| 层级 | 大小 | 用途 |
|------|------|------|
| H1 | 24px | 页面主标题 |
| H2 | 18px | 区块标题 |
| Body | 14px | 正文、列表项 |
| Small | 12px | 辅助信息、时间戳 |
| Button | 16px | 按钮文字 |

### 字色

| 用途 | 色值 |
|------|------|
| 正文 | `#19645c` |
| 辅助文字 | `#19645c` + 60% 透明度 |
| 按钮文字 | `#fef3ea`（深色底）/ `#19645c`（浅色底）|
| 链接 | `#84cccc` |

## 阴影规范

```css
/* 卡片阴影 */
.card-shadow {
  box-shadow: 0 2px 12px rgba(25, 100, 92, 0.08);
}

/* 按钮悬浮阴影 */
.btn-hover-shadow {
  box-shadow: 0 4px 16px rgba(204, 125, 125, 0.25);
}

/* 弹窗阴影 */
.modal-shadow {
  box-shadow: 0 8px 32px rgba(25, 100, 92, 0.12);
}
```

## 动画规范

| 动画 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| 按钮悬浮放大 | 200ms | ease-out | scale(1.05) |
| 按钮点击回弹 | 150ms | ease-in-out | scale(0.95) → scale(1) |
| 呼吸灯（录制中）| 1.5s 循环 | ease-in-out | opacity 0.4 ↔ 1.0 |
| 卡片入场 | 300ms | ease-out | translateY(20px) → 0 + opacity 淡入 |
| 弹窗出现 | 200ms | ease-out | scale(0.9) → 1 + opacity 淡入 |
| 开关切换 | 200ms | ease | 滑块平滑移动 |
| 进度条填充 | 300ms | ease-out | width 过渡 |

## 组件样式规范

### 主按钮 `.btn-primary`
```
背景: #cc7d7d
文字: #fef3ea
圆角: 20px
内边距: 12px 32px
字号: 16px
字体粗细: 600
悬浮: 背景变 #f5b4ae, scale(1.05), 阴影增强
按下: scale(0.95)
```

### 次要按钮 `.btn-secondary`
```
背景: 透明
边框: 2px solid #cc7d7d
文字: #cc7d7d
圆角: 20px
内边距: 10px 24px
悬浮: 背景 #fef3ea, 边框 #f5b4ae
```

### 卡片 `.card`
```
背景: #c4e4dd
圆角: 16px
内边距: 16px
阴影: card-shadow
悬浮: translateY(-2px), 阴影增强
```

### Toggle 开关 `.toggle`
```
关闭态: 背景 #d8eef1, 滑块 #fef3ea
开启态: 背景 #84cccc, 滑块 #fef3ea
尺寸: 宽 44px, 高 24px
滑块: 直径 20px 圆形
```

### 筛选标签 `.tab`
```
默认: 背景透明, 文字 #19645c
选中: 背景 #cc7d7d, 文字 #fef3ea
圆角: 10px
内边距: 6px 16px
```

### 右键菜单 `.context-menu`
```
背景: #fef3ea
圆角: 12px
阴影: modal-shadow
菜单项: 内边距 10px 20px
菜单项悬浮: 背景 #d8eef1
分隔线: 1px solid #c4e4dd
```

### 弹窗 `.modal`
```
遮罩: rgba(0,0,0,0.3)
内容: 背景 #fef3ea, 圆角 16px
标题: 18px, #19645c
按钮: 使用主/次要按钮样式
```
