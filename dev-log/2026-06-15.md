## 🐛 发现的问题及解决

### 问题1：Electron 下载失败
- **现象**：`npm install electron` 后二进制文件未下载，手动运行 install.js 报 `ECONNRESET`
- **原因**：GitHub CDN 被墙
- **解决**：设置 `ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"` 使用阿里云镜像

### 问题2：`require('electron')` 返回字符串而非 API 对象（关键问题）
- **现象**：`app.whenReady()` 报 `TypeError: Cannot read properties of undefined`
- **诊断**：`require('electron')` 返回了 `node_modules/electron/index.js` 导出的路径字符串，而非 Electron 内置模块
- **根因**：VS Code 终端设置了 `ELECTRON_RUN_AS_NODE=1`，导致 Electron 以纯 Node.js 模式运行，不提供内置 electron 模块
- **解决**：启动前 `unset ELECTRON_RUN_AS_NODE`（bash）或 `set ELECTRON_RUN_AS_NODE=`（cmd），已写入 `start.bat`
- **参考**：这是已知 Bug [electron/electron#49034](https://github.com/electron/electron/issues/49034)
